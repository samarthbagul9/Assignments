1. What is Java?

Answer:

Java is a class-based, high-level, object-oriented programming language.
It was developed by James Gosling.
It is used to build web, mobile, and enterprise applications and is platform independent due to JVM.


2. Is Java platform dependent or independent?

Answer:

Java is platform independent.
It follows WORA (Write Once, Run Anywhere).
Java code is converted to bytecode, which runs on JVM available on multiple systems.


3. What is JVM?

Answer:

JVM stands for Java Virtual Machine.
It is software that executes Java bytecode.
It also handles memory management and garbage collection.


4. What is JDK and JRE?

Answer:

JDK is used to develop and compile Java programs.
JRE provides the runtime environment to execute Java programs.
JRE includes JVM and necessary libraries.


5. Why Java is called WORA?

Answer:

Java code is compiled into bytecode.
Bytecode runs on any system with JVM.
Hence, it is called Write Once, Run Anywhere.


6. What is a class?

Answer:

A class is a blueprint or template.
It defines properties and behavior.
Objects are created using a class.


7. What is an object?

Answer:

An object is an instance of a class.
It represents real-world entities.
It contains data and methods.


8. What is main method?

Answer:

It is the entry point of a Java program.
Execution starts from the main method.
Without it, the program will not run.


9. What is System.out.println()?

Answer:

It is used to print output on console.
System.out is an output stream.
println() prints data and moves to the next line.


11. What are data types?

Answer:

Data types define the type of data a variable can store.
They help in proper memory allocation.
Example: int, float, char.


12. Examples of primitive data types?

Answer:

int, float, double, char, boolean.
These are basic built-in types.
They store simple values.


13. What is a variable?

Answer:

A variable is a container to store data.
It has a name and data type.
Its value can change during execution.


14. How do you declare a variable?

Answer:

By specifying data type and variable name.
Example:
int a;
It reserves memory for the variable.


15. Difference between int and float?

Answer:

int stores whole numbers.
float stores decimal numbers.
float requires f suffix.


16. What is type casting?

Answer:

Type casting is conversion of one data type into another.
It can be implicit or explicit.
Used when handling different data types.


17. Default value of int?

Answer:

Default value is 0.
Applies to instance variables.
Not applicable to local variables.


18. What is a keyword?

Answer:

A keyword is a reserved word in Java.
It has predefined meaning.
Example: int, class, if.


19. Can we use keyword as variable name?

Answer:

No, it is not allowed.
It results in a compile-time error.
Keywords are reserved by Java.


20. Difference between = and ==?

Answer:

= is an assignment operator.
== is a comparison operator.
== checks equality of values.


21. What is if statement?

Answer:

It is used for decision making.
Executes code based on condition.
Works with boolean expressions.


22. Difference between if and switch?

Answer:

if handles complex conditions.
switch handles fixed values.
switch is faster for multiple choices.


23. What is loop?

Answer:

Loop is used to repeat code.
It reduces redundancy.
Works based on condition.


24. Types of loops?

Answer:

for, while, do-while.
for → fixed iterations.
while → condition-based.


25. Difference between for and while?

Answer:

for is used when iterations are known.
while is used when unknown.
Syntax differs.


26. What is break?

Answer:

It terminates loop or switch.
Control moves outside block.
Used for early exit.


27. What is continue?

Answer:

It skips current iteration.
Moves to next iteration.
Loop continues execution.


28. Without main method?

Answer:

Program will not execute.
JVM looks for main method.
It is mandatory.


29. Can we run Java without class?

Answer:

No, Java is class-based.
Code must be inside class.
Execution requires class.


30. Without compilation?

Answer:

Program cannot run.
Compilation generates bytecode.
JVM executes bytecode.


31. Can variable change its type?

Answer:

No, data type is fixed.
It cannot change during execution.
Only value can change.


32. Size of int?

Answer:

int uses 4 bytes (32 bits).
It is a signed integer.
Range: -2³¹ to 2³¹-1.


33. Why Java does not support pointers?

Answer:

For security reasons.
To avoid direct memory access.
Makes Java simpler and safer.
